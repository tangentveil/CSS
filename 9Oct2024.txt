Problem Statement:
Objective: Create a simple HTML page containing three paragraphs, each with a heading. Additionally, apply specific styles to each paragraph using an external CSS file with id selectors.

Instructions:

HTML Structure:

Create an HTML file.
Add three paragraphs, each with its own heading.
Assign unique id attributes to the paragraphs: para-1, para-2, and para-3.
CSS Styling:

Create a CSS file.
Use id selectors to apply styles to the three paragraphs.
Apply the following CSS properties to each paragraph:
For #para-1:
Height, Width, Border radius, Border, Background color

For #para-2:
Background color, Height, Width, border top, border right, border bottom, border left, border top left radius: 4px, border top right radius border bottom left radius, border bottom right radius

For #para-3:
Height, Width, Background image, Border, Text color, background repeat, background position
